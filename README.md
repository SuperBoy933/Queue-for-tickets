# Queue-for-tickets
NEU Datastructure 
排队购票问题
【问题描述】
世界杯足球亚洲区预选赛正在激烈进行。决赛门票处于热卖。为使门票公平、安全的销售，售票处决定采用如下售票规则：
（1）购票者到购票处领取一个随机编号。购票者按随机编号从小到大排序。
（2）随机编号处于最小编号与最大编号之间的购票者，可直接到窗口排队购票。
（3）售票窗口空闲时随机发出0或1指令，指令为0时，最小编号者到窗口购票，指令为1时，最大编号者到窗口购票。
【设计要求】
设计算法实现按上述规则的排队售票程序。
（1）采用STL的双端队列类等数据结构。
（2）应用基本运算，设计算法求解。
